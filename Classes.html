Classes

    Since I have been in the program for almost a year, I have taken quite a few classes. Here are classes that I have taken or am currently taking with some reviews for each one.

Information Structure with Java

   This class is not a difficult one. I learned a lot about machine learning and object-oriented structures. Professor was really helpful in showing the processes.

Business Data Communication and Networks

   A foundation class of data communications and take a bottom-up approach to computer networks. The assignments and quizzes are very helpful to learn the material. The term project is time-consuming but doable. You will have fun converting binary numbers back and forth.

Database Design and Implementation for Business

   You will learn how to design a database system using SQL. It comes in really handy in the future. The term project is somewhat interesting. You will design either a database for hospital or for Netflix, so it will be fun to play around a bit.

Information Systems Analysis and Design

   A class about how to design and analyze the system. You will learn about System feasibility; requirements analysis; database utilization; Unified Modeling Language; software system architecture, design, and implementation, management; project control; and systems-level testing. There will be a group presentation.

IT Strategy and Management

   I will go ahead and say this class is a bit of a waste of time. Unless you are already in the manager position, this class will not contribute much to your understanding in information system.

Information Structures with Python

   Python is really important in Big Data. Python is actually really interesting to learn, and comparing to Java or C++, it is much simpler. My professor is not a very good at explaining stuffs, so it can be challenging quite a bit.

Quantitative Methods for Information Systems

   This is a pure math class. If you are good at Algebra and Calculus 1, then it will be a breez. I personally do not have any problem keeping up with the works. The professor is knowledgable and explains everything throughly.

Web Application Development

   A class about web developing. It is actually fun to learn HTML, CSS, and JavaScript.Maybe I have not reached the higher level, but this is definitely more relaxing than studying Python, Java, or C++. The term project is about creating your own websites. I do enjoy doing these, even though it can be a bit time-consuming.
